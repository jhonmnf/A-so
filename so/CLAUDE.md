# Claude Project Guide - Web Development

This project focuses on structured web development using a specialized set of skills and best practices.

## 🛠 Custom Skills
The AI should utilize the skills defined in `skills-mcp/my-web-skills/` to guide all web-related tasks. These skills provide the methodology and quality standards for this project.

### 1. Website Creation Workflow
- **Reference**: `skills-mcp/my-web-skills/website-creation/SKILL.md`
- **Usage**: Refer to this skill when starting a new website project. Follow the stages: Planning $\rightarrow$ Design $\rightarrow$ Tech Stack $\rightarrow$ Development $\rightarrow$ Testing $\rightarrow$ Deploy.

### 2. Web Best Practices Check-list
- **Reference**: `skills-mcp/my-web-skills/web-best-practices/SKILL.md`
- **Usage**: Refer to this skill during code reviews, audits, or final polish. Ensure the project meets all criteria for:
    - 🚀 Performance (Core Web Vitals, Lazy Loading)
    - 🔍 SEO (Semantic HTML, Meta Tags)
    - ♿ Accessibility (WCAG, Keyboard Nav)
    - 🛡️ Security (HTTPS, Input Sanitization)
    - 📱 UX (Mobile-First, Visual Hierarchy)

## 📜 Development Guidelines
- Always prioritize semantic HTML.
- Follow a mobile-first approach for all CSS/Layouts.
- Ensure all interactive elements are accessible via keyboard.
- Optimize images to WebP/AVIF format before deployment.
